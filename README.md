# To Do Application

This app helps users to track their todos

#### Intsructions
1. Download the source code
```bash 
git clone https://github.com/zarifboev-abdulaziz/ToDoAppNodeJs.git
```
or just download the zip file

2. Install deps

```bash
npm install 
```

3. Run the app
```bash
node app
```

#### Dependency list 
- express
- pug
